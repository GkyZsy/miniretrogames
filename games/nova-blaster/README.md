# Nova Blaster

Nova Blaster is a small Mega Man-inspired 2D action platformer built with **HTML, CSS, and vanilla JavaScript**.

## Features

- Side-scrolling 2D platform action
- Run, jump, shoot, and dash controls
- Three compact stages
- Patrol enemies, turrets, flyers, and a final boss
- Health pickups and score tracking
- Best score stored with `localStorage`
- No libraries required

## Controls

- **Move:** `A / D` or `Left / Right`
- **Jump:** `W`, `Up Arrow`, or `Space`
- **Shoot:** `J` or `X`
- **Dash:** `K` or `C`
- **Pause:** `P`
- **Restart:** `R` after game over or victory

## How to Run

Just open `index.html` in your browser.

For the best experience, use a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
