# MiniRetroGames Ready Pack

This package includes:

- a polished homepage with 6 game cards
- the user's **Nova Blaster** game
- 5 additional lightweight browser games
- GitHub Pages friendly relative links

## Structure

```text
miniretrogames-ready/
├── index.html
├── style.css
├── script.js
└── games/
    ├── nova-blaster/
    ├── flappy-birb/
    ├── star-blaster/
    ├── brick-breaker/
    ├── tiny-snake/
    └── dash-runner/
```

## Publish

Upload the contents of this folder to your `miniretrogames` repository root.
